# Feature: Inventory Check & Store Lookup

**ID**: F-04 / F-05
**Requirements**: FR-04, FR-05
**Use Cases**: UC-04, UC-05

## Summary

As an MCP client, I want to check whether a product is in stock at a specific store
and find nearby Home Depot locations so that I can help users plan their trip.

## Scope

- In-store inventory check by item ID + store ID
- Aisle and bay location data when available
- Store lookup by ZIP code with distance and hours
- Zip-to-store resolution for inventory checks

## Out of Scope

- Online order tracking
- Ship-to-home availability details beyond "In Stock / Out of Stock"
