# Feature: Product Search

**ID**: F-01
**Requirement**: FR-01
**Use Cases**: UC-01, UC-06, UC-07

## Summary

As an MCP client, I want to search Home Depot's product catalog by keyword so that
I can surface relevant products for a user's home improvement needs.

## Scope

- Keyword-based product search
- Optional category filter
- Optional sort order
- Pagination support

## Out of Scope

- Cart or purchase actions
- Price history or deal tracking
- Cross-retailer search
